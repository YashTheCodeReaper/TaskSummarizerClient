Navigate to your Atlassian JIRA settings and find Atlassian account settings and click that.

Find the security tab in your Atlassian account settings page.

Find the api tokens section and click create and manage api tokens.

Click create api token and a popup will appear.

Now choose a label for your api token. This label is finalized and it cannot be modified later.

Now you will get your api token. Please note that it will disappear after you close the popup. So make sure you copied and saved it somewhere.
